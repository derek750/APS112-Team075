# Team075 Tools

Tools we used for visuals, for MoS and final presentation

Project: Optimizing the Chemical Engineering Common Room

## Widgets

### Route Heatmap Visualizer (`route heatmap widget.html`)

A route heatmap visualizer of routes taken by students within the room

### Conflict Rate Calculator (`conflict calc widget.html`)

Calculator for the current conflicts in the room, and uses the model we created to calculate a predicted conflict amount for our new room design
